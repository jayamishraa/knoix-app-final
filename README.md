This is the submission for KoinX Frontend Intern Assignment.

Here is the deployed link: https://knoix-app-final.vercel.app/bitcoin

The project was built using NextJS and TS.

To run the project locally type these commands in the terminal:
   npm i
   npm run dev

Here is an overview of the completed tasks:

Mandatory Tasks:

1. Figma Design Implementation:
   - Successfully implemented all the components as per the provided Figma design using Next.js.

2. Coingecko API Integration:
   - Utilized Coingecko’s /simple/price API to fetch the price of Bitcoin in USD and INR as specified in the job description.

3. TradingView Chart Integration:
   - Integrated TradingView’s BTCUSD chart widget in place of the chart component in the Figma design, closely resembling the provided design.

4. Trending Coins Section:
   - Utilized Coingecko’s /search/trending API to display the top 3 trending coins in the "Trending Coins (24h)" component.

5. You May Also Like Section:
   - Implemented a horizontally scrollable carousel for the "You May Also Like" section, displaying logos, symbols, prices, price changes, and price graphs of trending coins.

6. Responsive UI:
   - Ensured the UI is responsive, aligning with the specifications outlined in the Figma design.

7. GitHub Repository:
   - Hosted the code on GitHub, adhering to best practices and maintaining clean code structure.

8. Deployment:
   - Deployed the code using Varcel, ensuring accessibility and optimal performance.

Optional Task:

- Dynamic Token Display:
  - Implemented the optional task by making the token shown dynamic using the URL. For instance, /bitcoin displays Bitcoin’s data and chart, while /ethereum shows Ethereum’s data and chart.
